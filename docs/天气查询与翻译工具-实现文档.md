## 二、外部 API 方案

### 2.1 天气查询 - wttr.in

- **接口地址**: `https://wttr.in/{城市名称}?format=j1`
- **请求方式**: GET
- **认证**: 无需 API Key
- **返回格式**: JSON

**响应数据结构要点**:

| 字段                                        | 说明                 |
| ------------------------------------------- | -------------------- |
| `current_condition[0].temp_C`               | 当前温度（摄氏度）   |
| `current_condition[0].humidity`             | 湿度（%）            |
| `current_condition[0].weatherDesc[0].value` | 天气描述（如 Sunny） |
| `current_condition[0].windSpeedKmph`        | 风速（km/h）         |
| `current_condition[0].feelsLikeC`           | 体感温度             |
| `current_condition[0].visibility`           | 能见度（km）         |
| `current_condition[0].weatherCode`          | 天气代码             |
| `nearest_area[0].areaName[0].value`         | 地区名称             |
| `nearest_area[0].country[0].value`          | 国家                 |

**请求示例**:

```
GET https://wttr.in/Beijing?format=j1
```

**特点**:

- 支持中文城市名（如 `北京`）
- 支持英文城市名（如 `London`）
- 免费、无速率限制（合理使用即可）

### 2.2 文本翻译 - MyMemory API

- **接口地址**: `https://api.mymemory.translated.net/get?q={文本}&langpair={源语言}|{目标语言}`
- **请求方式**: GET
- **认证**: 无需 API Key
- **返回格式**: JSON
- **免费额度**: 约 1000 次/天（无 Key 情况下）

**响应数据结构要点**:

| 字段                          | 说明               |
| ----------------------------- | ------------------ |
| `responseData.translatedText` | 翻译结果文本       |
| `responseData.match`          | 匹配度（1-100）    |
| `responseStatus`              | 状态码（200 成功） |

**请求示例**:

```
GET https://api.mymemory.translated.net/get?q=Hello%20World&langpair=en|zh-CN
```

**支持的语言代码**（常用）:

| 语言         | 代码  |
| ------------ | ----- |
| 英语         | en    |
| 中文（简体） | zh-CN |
| 中文（繁体） | zh-TW |
| 日语         | ja    |
| 韩语         | ko    |
| 法语         | fr    |
| 德语         | de    |
| 西班牙语     | es    |
| 俄语         | ru    |
| 阿拉伯语     | ar    |
| 葡萄牙语     | pt    |
| 意大利语     | it    |
| 荷兰语       | nl    |

## 三、后端路由设计

### 3.1 天气查询路由（`routes/weather.js`）

| 方法 | 路径                 | 功能                 | 请求参数                               |
| ---- | -------------------- | -------------------- | -------------------------------------- |
| GET  | `/api/weather/:city` | 查询指定城市天气     | `city`: 城市名（路径参数，支持中英文） |
| GET  | `/api/weather`       | 根据 IP 查询当地天气 | 无参数（默认查询北京）                 |

**成功响应格式**:

```json
{
  "success": true,
  "data": {
    "city": "Beijing",
    "country": "China",
    "temperature": "22",
    "feelsLike": "20",
    "humidity": "45",
    "weatherDesc": "Sunny",
    "windSpeed": "15",
    "visibility": "10",
    "weatherCode": 113
  }
}
```

**错误响应格式**:

```json
{
  "success": false,
  "error": "未找到该城市的天气信息"
}
```

### 3.2 翻译路由（`routes/translate.js`）

| 方法 | 路径             | 功能     | 请求参数                                                           |
| ---- | ---------------- | -------- | ------------------------------------------------------------------ |
| POST | `/api/translate` | 翻译文本 | Body（JSON）: `{ "q": "文本", "source": "en", "target": "zh-CN" }` |

**成功响应格式**:

```json
{
  "success": true,
  "data": {
    "translatedText": "你好，世界",
    "source": "en",
    "target": "zh-CN",
    "match": 100
  }
}
```

**错误响应格式**:

```json
{
  "success": false,
  "error": "翻译服务暂时不可用，请稍后重试"
}
```

## 四、Service 层设计

### 4.1 `services/weatherService.js`

**职责**: 封装对 wttr.in 的 HTTP 请求，解析响应并提取关键字段。

**核心函数**:

```javascript
/**
 * 根据城市名查询天气
 * @param {string} city - 城市名称（支持中英文）
 * @returns {Promise<Object>} 格式化的天气数据
 */
async function getWeatherByCity(city) { ... }
```

**实现要点**:

- 使用 Node.js 18+ 内置的 `fetch` 发起请求（无需额外安装 `node-fetch`）
- 设置 5 秒超时，超时抛出友好错误
- 对 wttr.in 返回的 JSON 进行字段提取和转换
- 对城市名进行 `encodeURIComponent` 编码
- 异常捕获：网络错误、JSON 解析错误、空数据

### 4.2 `services/translateService.js`

**职责**: 封装对 MyMemory API 的 HTTP 请求，解析翻译结果。

**核心函数**:

```javascript
/**
 * 翻译文本
 * @param {string} text - 待翻译文本
 * @param {string} source - 源语言代码
 * @param {string} target - 目标语言代码
 * @returns {Promise<Object>} 翻译结果数据
 */
async function translateText(text, source, target) { ... }
```

**实现要点**:

- 使用 Node.js 18+ 内置的 `fetch`
- 对文本和语言参数进行 URL 编码
- 校验文本非空、语言代码合法
- 设置 5 秒超时
- 异常捕获：网络错误、API 非 200 响应

## 六、前端与后端交互说明

- **静态文件托管**: 使用 `express.static('public')` 中间件
- **数据交换格式**: 全部使用 JSON
- **前端请求**: 使用浏览器原生 `fetch` API（无需引入额外库）
- **页面跳转**: 使用 `<a>` 标签或 `window.location.href` 跳转

**请求流程图**:

```
浏览器 (index.html)
    │
    ├── 点击"天气查询" ──→ weather.html
    │                           │
    │                   输入城市名，点击查询
    │                           │
    │                    fetch GET /api/weather/北京
    │                           │
    │                   后端 routes/weather.js
    │                           │
    │                   weatherService.getWeatherByCity("北京")
    │                           │
    │                    fetch https://wttr.in/北京?format=j1
    │                           │
    │                   ←── 返回 JSON 天气数据 ──→ 渲染页面
    │
    └── 点击"翻译工具" ──→ translate.html
                            │
                    输入文本，选择语言，点击翻译
                            │
                     fetch POST /api/translate (body: {q, source, target})
                            │
                    后端 routes/translate.js
                            │
                    translateService.translateText(text, source, target)
                            │
                     fetch https://api.mymemory.translated.net/get?q=...&langpair=...
                            │
                    ←── 返回 JSON 翻译结果 ──→ 渲染页面
```

## 八、`app.js` 修改要点

需要在现有 `app.js` 中新增以下内容（保留原有 Todo 代码不变）：

```javascript
// 托管静态文件（新增）
app.use(express.static("public"));

// 引入路由模块（新增）
const weatherRouter = require("./routes/weather");
const translateRouter = require("./routes/translate");

// 挂载路由（新增）
app.use("/api/weather", weatherRouter);
app.use("/api/translate", translateRouter);
```

## 九、错误处理策略

| 场景                | 处理方式                                             |
| ------------------- | ---------------------------------------------------- |
| API 请求超时        | 设置 5 秒超时，超时返回 "服务响应超时，请稍后重试"   |
| 城市不存在          | wttr.in 返回空数据时，返回 "未找到该城市的天气信息"  |
| 翻译文本为空        | 后端校验文本参数，返回 "请输入待翻译的文本"          |
| MyMemory 服务不可用 | 非 200 响应时，返回 "翻译服务暂时不可用，请稍后重试" |
| 网络错误            | 捕获异常，返回通用错误 "服务异常，请检查网络后重试"  |
| 输入非法字符        | 对参数进行编码，防止注入攻击                         |
